¿Cómo participar?

¿Qué pasos debemos dar?

Si quieres partiripar en el proyecto solamente necesitas:

  1. Forma un equipo de 7 estudiantes
  2. Selecciona un proyeto
  3. Desarrolla un proyecto para nuestros abuelos
  4. Comparte lo creado

Ejemplo de productos finales

¿Qué podemos construir?

¿Sabes cómo se hacen los paparajotes? ¿Lo saben tus abuelos? ¿Cuánta valiosa información poseen nuestros abuelos y nosotros podemos documentar?

Podemos crear:

  1. Un vídeo explicativo.
  2. Un documento.
  3. Un cartel con la información recogida.

Ejemplo de productos finales

Y si construimos algo para ayudar

¿Qué podemos construir para ayudar a nuestros abuelos? Imagina una aplicación para localizar las llaves, la cartera... ¿Qué otras ideas se te ocurren?

Podemos desarrollar:

  1. Ideas para programar.
  2. Crear la aplicación.
  3. Incluso, construir un robot.
